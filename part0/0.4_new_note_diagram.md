# Exercise 0.4: New Note Diagram

```mermaid
sequenceDiagram
    participant U as User
    participant B as Browser
    participant S as Server

    Note over U,B: User types text and clicks Save (form submit)

    B->>S: POST /exampleapp/new_note
    S-->>B: 302 Found (redirect to /notes)

    B->>S: GET /exampleapp/notes
    S-->>B: HTML document

    B->>S: GET /exampleapp/main.css
    S-->>B: CSS file

    B->>S: GET /exampleapp/main.js
    S-->>B: JavaScript file

    Note right of B: Browser runs JS and fetches notes JSON

    B->>S: GET /exampleapp/data.json
    S-->>B: [ { "content": "...", "date": "..." }, ... ]

    Note right of B: Browser renders updated notes list
