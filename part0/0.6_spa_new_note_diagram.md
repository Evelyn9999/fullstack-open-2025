# Exercise 0.6: New Note in SPA

```mermaid
sequenceDiagram
    participant U as User
    participant B as Browser spa.js
    participant S as Server

    Note over U,B: User types text and clicks Save

    %% JS intercepts the submit (no page reload)
    B->>B: onsubmit handler (preventDefault)
    B->>B: create note object
    B->>B: push note to array
    B->>B: redraw notes

    Note right of B: DOM updates immediately (no reload)

    %% Send the note to the server as JSON
    B->>S: POST /exampleapp/new_note_spa (application/json)
    activate S
    S-->>B: 201 Created (no redirect)
    deactivate S

    Note right of B: Page stays as-is. No GET /notes, no reload.
